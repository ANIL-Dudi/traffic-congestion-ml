# 🚦 Traffic Congestion Prediction using Computer Vision

## 📌 Overview
This project uses computer vision and machine learning to predict traffic congestion levels from video data.

## 🎯 Problem Statement
Urban traffic congestion leads to delays, fuel wastage, and inefficiencies.  
This project aims to automate traffic monitoring using AI.

## ⚙️ Tech Stack
- Python
- OpenCV
- YOLOv5
- Matplotlib

## 🚀 Features
- Real-time vehicle detection
- Frame-wise congestion estimation
- Data visualization of traffic patterns

## 🧠 Approach
1. Capture video input
2. Detect vehicles using YOLOv5
3. Count vehicles per frame
4. Estimate congestion levels
5. Visualize trends
